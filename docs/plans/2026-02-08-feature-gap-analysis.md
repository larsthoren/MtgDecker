# MtgDecker Feature Gap Analysis

> Comparison against Moxfield, Archidekt, Manabox, MTGGoldfish, TappedOut, Deckstats

## HIGH PRIORITY (present in nearly all competitors)

1. **Card Pricing** — Per-card prices from Scryfall, deck totals, collection value
2. **Sample Hand / Playtesting** — Random 7-card hand, London mulligan, draw next
3. **Maybeboard** — "Considering" zone alongside MainDeck and Sideboard
4. **Extended Deck Stats** — Average CMC, land-to-spell ratio, mana source analysis, rarity breakdown
5. **More Export Formats** — Plain text, CSV, clipboard copy, Moxfield/Archidekt compatibility

## MEDIUM PRIORITY

6. Visual Deck View (card image grid in deck builder, hover previews)
7. Draw Probability Calculator (hypergeometric distribution)
8. Custom Card Tags ("Removal", "Ramp", "Win Con" per card in deck)
9. Commander-Specific Features (commander zone, color identity validation)
10. Deck Folders / Organization (folders, tags, sort/filter)
11. "Build from Collection" Filter (toggle owned-only in search)

## LOWER PRIORITY

12. Proxy printing (PDF generation)
13. Deck comparison (side-by-side)
14. Deck versioning/history
15. Wishlist/Acquireboard
16. Keyboard shortcuts
17. Drag and drop
18. Card rulings display
19. Standard + Pioneer format support
20. Collection import (CSV, Manabox, Deckbox)
